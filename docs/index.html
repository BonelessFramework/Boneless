    button

    @mixin button($classname: btn, $size: medium, $color: #888, $type: border, $rounded: true) { ... }
    View source

    Description

    Used to create button styles

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $classname

    classname of button styles

    		Stringbtn
    $size

    size of button can be small, medium or large

    		Stringmedium
    $color

    color of button

    		Color#888
    $type

    style of button can be border, solid

    		Stringborder
    $rounded

    to declare whether the button has rounded corners, if a number is passed corners are rounded by that amount (number must be a px value), if the word 'pill' is passed creates a pill style button

    		Boolean or Number or Stringtrue

    Example

    Example mixin usage

      @include button($classname: 'button', $color: #999);

    Usage of mixin to style button and input[type="submit"] elements

      button,
  input[type="submit"] {
    @include button($classname: false, $color: #999);
  }

    Usage of mixin creating rectangular buttons

      @include button($classname: button-flat, $color: #2ecc40, $rounded: false);

    Usage of mixin creating pill style buttons

      @include button($classname: button-pill, $color: #39cccc, $rounded: pill);

    • Benjamin Hinchley
